## Security

GrimoireRandom is not designed for cryptographic use and should not be used in security-sensitive contexts. It is intended for general-purpose randomness in games and similar applications.

If you discover a security vulnerability in Grimoire, please consult the [SECURITY.md](SECURITY.md) guidelines for responsible disclosure.
